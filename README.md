**Laravel NextJS Template**
=====================================

This is a boilerplate for Laravel with Passport and NextJS integration, including authentication implementation.

### Available API Routes

1. **Retrieve Access Token and Refresh Token**: `/oauth/token`
2. **Retrieve Authenticated User Information**: `api/user`

Note: This template provides a basic setup for authentication using Laravel Passport and NextJS. You can extend and customize it as per your project requirements.
